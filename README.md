# MedConnect SL - Landing Page

A modern, responsive landing page for **MedConnect SL**, a digital healthcare platform connecting Sierra Leoneans to quality healthcare services.

## 📋 Project Overview

MedConnect SL is a web-based platform designed to improve healthcare accessibility in Sierra Leone by:
- Connecting patients to nearby hospitals and clinics
- Enabling appointment booking
- Providing health tips and information
- Offering emergency contact access

## 🎯 Features

- **Find Hospital/Clinic** - Locate nearby healthcare facilities
- **Book Appointment** - Schedule appointments with healthcare providers
- **Health Tips** - Access useful health information and resources
- **Emergency Access** - Quick access to emergency contacts
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices

## 🚀 Getting Started

1. **Open the Landing Page**
   - Simply open `index.html` in your web browser

2. **View the Page**
   - The landing page displays with all sections: header, hero, services, and footer

3. **Navigation**
   - Use the navigation menu to explore different sections
   - Click "Find Hospital" or "Book Appointment" buttons for CTA actions

## 📱 Responsive Design

The landing page is designed to be responsive and adapts to different screen sizes:
- Desktop (1024px and above)
- Tablet (768px - 1023px)
- Mobile (below 768px)
